main.c
	En este archivo se encuentra el tercer número más grande de un vector de determnado tamaño. Se encuentra este número de dos maneras, una en donde se modifica el vector original y otra en donde no se modifica el vector original. Además de esto se incluya la función de encontrar el máximo.


iii)
salida_txt.c
	En este programa se le pide al usuario que ingese ciertos datos y se graban en un archivo de texto plano

	Hace lo mismo que el anterior, solo que este guarda los datos en un archivo binario.

iv)
El archivo de texto plano muestra el texto con caracteres de la tabla ascii. Esto es que puede ser leído fácilmente por una persona. Mientras tanto, el archivo binario muestro como es que la máquina interpreta en binario las variables y valores guardados.

v)
aleatorios.c
Aquí se generan datos aleatorios entre -10 y 200, que son flotantes y estos datos se guardan en un archivo binario.

vi)
Este programa lee los datos generados anteriormente y obtiene la varianza, el promedio y el máximo elemento.
