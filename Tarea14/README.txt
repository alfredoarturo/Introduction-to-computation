Se definen funciones que crean matrices de dos o tres dimensiones.
EStas funciones reciben un doble aputador para la de dos dimensiones, un triple a puntador para la de tres dimensiones junto con las dimensiones de las matrices a generar.
Además hay dos funciones que regresan el apuntador con la matriz y solo se ingersan las dimensiones.
LAs matrices ademaás se pueden llanar aleatoriamente con una funcion y se imprimen en la terminal.
